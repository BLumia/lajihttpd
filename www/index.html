Lajihttpd Default Page

It Works

This is the default welcome page used to test the correct operation of the lajihttpd server after installation on your systems. If you can read this page, it means that the lajihttpd HTTP server installed at this site is working properly. You should replace this file (located at the path you defined in the configuration file httpd.conf with variable name WWW_PATH) before continuing to operate your HTTP server.

Last paragraph is copy and edit from Apache's It Works page.
